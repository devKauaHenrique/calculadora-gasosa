# ⛽ Calculadora Gasosa

Uma aplicação web desenvolvida com React e TypeScript para auxiliar na comparação entre álcool e gasolina, indicando qual combustível oferece o melhor custo-benefício para abastecimento.

O projeto foi desenvolvido como parte dos estudos em React JS e TypeScript, com foco na manipulação de estados, formulários e lógica de negócios.

## 🚀 Demonstração

🔗 Acesse o projeto online:

https://calculadora-gasosa-six.vercel.app

## ✨ Funcionalidades

* Comparação entre álcool e gasolina
* Cálculo automático do melhor combustível
* Interface intuitiva e responsiva
* Validação dos campos informados
* Resultado exibido em tempo real
* Experiência otimizada para dispositivos móveis

## 💻 Como Executar Localmente

1. Clone o repositório

```bash
git clone https://github.com/devKauaHenrique/calculadora-gasosa.git
```

2. Entre na pasta do projeto

```bash
cd calculadora-gasosa
```

3. Instale as dependências

```bash
npm install
```

4. Execute a aplicação

```bash
npm run dev
```

5. Abra no navegador

```bash
http://localhost:5173
```
